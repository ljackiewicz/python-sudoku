# Python Sudoku Solver and Generator

## Description
Project prepared for educational purposes. It is intended to help expand and organize the current knowledge related to Python.

## Usage
#### Running project
```shell
python3 sudoku.py

# or

chmod u+x sudoku.py
./sudoku.py
```

#### Running unit tests
```shell
python3 -m unittest -v
```
