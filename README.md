## Usage
#### Preparing an environment (virtualenv)
```shell
# creating virtual environment (venv)
virtualenv venv

# activating virtual environment
source venv/bin/activate

# installing requirements
pip install -r requirements.txt
```

#### Running project
```shell
python3 sudoku.py

# or

chmod u+x sudoku.py
./sudoku.py
```

#### Running unit tests
```shell
python3 -m unittest -v
```
